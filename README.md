# Claude Clicker

A Cookie Clicker variant starring the Claude Code pixel mascot. Single-file HTML, no build step.

## Features

- **Pixel-art mascot** with click squish, crit clicks, and floating sparkles
- **18 upgrades** across two shops: Auto (11) and Click (7)
- **Combo system** — chain clicks for bonus multipliers
- **Golden Claude** spawns every 1–3 min → Frenzy (×7 for 15s) or Lucky (instant tokens)
- **Prestige (Vibes)** — ascend at 1B+ all-time tokens for permanent +1%-per-Vibe boost
- **31 achievements** with toast notifications
- **Offline progress** — earn at 50% rate while away (8h cap)
- **Buy ×1 / ×10 / ×100 / Max** bulk-purchase toggle
- **Auto-save** to localStorage every 5s + on unload
- **Keyboard shortcuts**: `Space` to click, `1/2/3/M` for buy multipliers
- **News ticker**, **progress bars**, and live stat chips

## Run locally

Just open `index.html` in a browser. No build, no dependencies.
